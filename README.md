# Présentation

TraderBook IDE est un éditeur de code simple orienté plugins.

# Installer l'application

```bash
git clone git@github.com:nico-develop/traderbook-ide.git
cd traderbook-ide

npm i

npm run build

npm start
```

# Compilation

Création des différents exécutables.

```bash
npm run dist
```
